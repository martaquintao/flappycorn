# 🦄 Flappycorn

A Flappy Bird clone with a twist — you play as a cute unicorn flying through rainbow curtains and flower stems.

Built entirely with vanilla HTML5 Canvas. No dependencies, no build step.

## How to play

Download or clone the repo, then open `flappycorn.html` in any browser.

```bash
git clone https://github.com/martaquintao/flappycorn.git
cd flappycorn
open flappycorn.html
```

- **Space** / **click** / **tap** — flap
- Fly through the gap between the rainbow (top) and the flower (bottom)
- Hit anything and it's game over
- Your best score is saved between sessions
